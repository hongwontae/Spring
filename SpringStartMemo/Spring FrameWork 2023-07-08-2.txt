여러 유형의 의존성 주입

의존성 주입에는 3개의 유형이 있다.
1. 생성자 기반
2. 수정자 기반(Setter)
3. 필드 기반

컴포넌트 스캔은 아무것도 명시하지 않으면 현재 패키지에서 컴포넌트 스캔을 자동으로 수행한다.

생성자 기반 주입은 한번에 초기화를 하기 떄문에 안전하고 유용하다.

의존성 주입
=> 어떤 객체가 사용하는 의존 객체를 직접 만들어 사용하는게 아니라 주입 받아 사용하는 것이다.
어렵게 생각하지 말고 new 연산자로 객체를 만든다고 생각하면 된다.

@Component => 클래스에 사용

@Bean => 메소드에 사용

Spring 프레임워크는 객체의 생명 주기를 관리
@Autowired => 의존성을 표시하는 어노테이션
